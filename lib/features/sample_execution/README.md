# Sample Execution Entry Feature

## Overview
Mobile-optimized sample execution entry screen for RAK White Cement app with search functionality and data table.

## Files Created

### 1. Screen
- `sample_execution_entry_screen.dart` - Main mobile UI screen

### 2. Service
- `lib/core/services/sampling_execution_service.dart` - API integration

### 3. Models
- `lib/core/models/sampling_drive_models.dart` - Data models

## Features

✅ Mobile-first responsive design with ScreenUtil
✅ Smooth animations (fade, slide)
✅ Search functionality (retailer code, name, painter name, date range)
✅ Top 100 entries loaded on init with fallback
✅ Interactive data table (tap row to populate form)
✅ Form validation with required field indicators
✅ Emirates dropdown with fallback data
✅ Date pickers for all date fields
✅ File upload for sample photograph
✅ Reimbursement mode with auto-fill logic
✅ Success/error notifications
✅ Modern Material Design 3 UI

## API Integration

### Endpoints

#### GET `/api/SamplingExecutionEntry/top100`
Returns the latest 100 sampling execution entries.

**Response:** Envelope format
```json
{
  "success": true,
  "data": [
    {
      "docuNumb": "SEE-DUB-00123",
      "retailerName": "ABC Store",
      "retailerCode": "RET001",
      "distributorName": "XYZ Distributor",
      "emirates": "DUB",
      "distributionDate": "2024-01-15T00:00:00",
      "painterName": "John Doe",
      "painterMobile": "501234567",
      "qtyDistributedKg": 25.5,
      "siteAddress": "123 Main St",
      "sampleDate": "2024-01-15T00:00:00",
      "product": "Wallcare Putty",
      "photoImage": null,
      "reimbursementMode": "Max 150 AED",
      "reimbursementAmountAED": 150.0,
      "sampleCancelFlag": "S"
    }
  ],
  "error": null
}
```

#### GET `/api/SamplingExecutionEntry/search`
Search entries with filters.

**Query Parameters:**
- `retailerCode` - Filter by retailer code
- `retailerName` - Filter by retailer name
- `painterNameFilter` - Filter by painter name
- `start` - Start date (yyyy-MM-dd)
- `end` - End date (yyyy-MM-dd)
- `top` - Limit results (default: 200)

**Response:** Same envelope format as top100

#### POST `/api/SamplingExecutionEntry/submit`
Submit a new sampling execution entry.

**Request Body:**
```json
{
  "retailerName": "ABC Store",
  "retailerCode": "RET001",
  "distributorName": "XYZ Distributor",
  "emirates": "DUB",
  "distributionDate": "2024-01-15",
  "painterName": "John Doe",
  "painterMobile": "501234567",
  "qtyDistributedKg": 25.5,
  "siteAddress": "123 Main St",
  "sampleDate": "2024-01-15T00:00:00",
  "product": "Wallcare Putty",
  "photoImage": "base64_encoded_image",
  "reimbursementMode": "Max 150 AED",
  "reimbursementAmountAED": 150.0,
  "sampleCancelFlag": "S"
}
```

**Response:**
```json
{
  "success": true,
  "data": "SEE-DUB-00123",
  "error": null
}
```

## Form Fields

### Sample Material Distribution
- Retailer Name (required)
- Retailer Code (required)
- Concern Distributor (required)
- Emirates (dropdown, required)
- Date of Distribution (required)

### Execution Details
- Painter/Contractor Name (required)
- Contact Number (required, UAE format)
- Material Qty Distributed in Kg (required)
- Site Address (optional)

### Sample Proof
- Sample Date (optional)
- Product (dropdown: Wallcare Putty)
- Sample Photograph (upload, jpg/jpeg/png, max 10MB)

### Reimbursement
- Reimbursement Mode (dropdown: 0 AED or Max 150 AED)
- Amount Reimbursed (auto-filled, editable for Max 150 AED)

## Data Table Features

- Horizontal scroll for mobile
- Shows: Retailer, Code, Distributor, Emirates, Date, Painter, Mobile, Distributed Kg
- Tap any row to populate form fields
- Search/filter functionality
- Shows entry count

## States

1. **Loading** - Shows spinner while fetching top 100
2. **Data Loaded** - Shows entry count and table
3. **Search Results** - Filtered entries
4. **No Results** - When search returns empty
5. **Error** - When API fails
6. **Empty** - No entries available
