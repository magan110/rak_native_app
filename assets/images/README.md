## Logo Usage

The `DualLogoWidget` is used throughout the app to display both logos consistently:
- Supports circular and rectangular display
- Responsive sizing
- Configurable spacing between logos
- Error handling with fallback icons

## Replacing Logos

If you need to replace either logo:
- RAK logo: Save as `rak_logo.jpg` in this directory
- Birla logo: Save as `birla_logo.png` in this directory
- Supported formats: JPG, PNG
- Recommended size: 512x512 pixels or higher for crisp display
